# IntelliJ Navigation & Error Fix Cheat Sheet

## 🔎 Go to Class / Code Navigation

### Go to Declaration
- Windows/Linux: `Ctrl + B`
- Mac: `Cmd + B`
- Or: `Ctrl/Cmd + Click`

👉 Jumps to where a class, method, or variable is defined.

---

### Go to Implementation (important for interfaces)
- Windows/Linux: `Ctrl + Alt + B`
- Mac: `Cmd + Alt + B`

👉 Jumps to the actual implementing class.

---

### Search / Open Anything

#### Search Class
- Windows/Linux: `Ctrl + N`
- Mac: `Cmd + O`

#### Search File
- Windows/Linux: `Ctrl + Shift + N`
- Mac: `Cmd + Shift + O`

#### Search Everywhere
- `Shift + Shift`

---

## 🔁 Navigate Through Code

### Recent Files
- Windows/Linux: `Ctrl + E`
- Mac: `Cmd + E`

### Go Back / Forward
- Back: `Ctrl + Alt + Left`
- Forward: `Ctrl + Alt + Right`

### File Structure (methods in current file)
- Windows/Linux: `Ctrl + F12`

---

## ❌ Errors & Debugging Navigation

### Jump Between Errors
- Next error: `F2`
- Previous error: `Shift + F2`

👉 Cycles through all errors in the file/project.

---

### Show All Problems
- Windows/Linux: `Alt + 6`
- Mac: `Cmd + 6`

👉 Opens Problems panel (all errors + warnings).

---

### Quick Fix
- `Alt + Enter`

👉 Shows IntelliJ suggestions to fix the current error.

---

## ✏️ Editing Speed Shortcuts

### Rename Symbol (VERY important)
- Windows/Linux: `Shift + F6`

### Duplicate Line
- Windows/Linux: `Ctrl + D`

### Delete Line
- Windows/Linux: `Ctrl + Y`

### Move Line Up/Down
- Windows/Linux: `Shift + Alt + Up/Down`

---

## ⚡ Fast Workflow for Fixing Errors

1. Press `F2` → jump to first error
2. Press `Alt + Enter` → fix it
3. Press `F2` again → next error
4. Repeat until clean

---

## 🧠 Mental Model

- `Ctrl + B` → teleport to definition
- `F2` → cycle through problems
- `Alt + Enter` → fix suggestions
- `Ctrl + E` → jump back to recent work
